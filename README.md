# book-search
# Description
This is a book search engine that allows users to search for books, create an account, save books to their account, and manage their saved books. The search engine integrates with the Google Books API to provide accurate search results.

# Installation
To install necessary dependencies, run the following command: Run npm i to install the dependencies. Use npm run build to build the site. Then run npm start to start the server.
# Usage
1. The main menu will be displayed, showing options to "Search for Books" and "Login/Signup."
2. To search for books, click on the "Search for Books" menu option or enter a search term in the input field and click the submit button.
3. To create an account or log in, click on the "Login/Signup" menu option.-
4. Perform a book search, and the search results will include a "Save" button to add books to your account.
5. To view your saved books, click on the "My Saved Books" option.
6. All your saved books will be displayed with their details and a link to the Google Books site. You can also remove books from this list.
7. To log out, click on the "Logout" option.
